# Yeda Video Player

## Run the server:
```bash
npm install
```
```bash
npm run dev
```

### Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Keyboard Controls:
### Space - Play/Pause
### Left/Right Arrows - Skip 10 seconds
### Up/Down Arrows - Volume control
### M - Mute/Unmute
### F - Toggle fullScreen
### <> - Playback speed control
